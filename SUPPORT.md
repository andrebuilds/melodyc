# Support

## Getting Help

For setup and usage questions, start with the project documentation:

- [README](README.md)
- [Frontend getting started guide](frontend/getting-started.md)
- [Backend getting started guide](backend/getting-started.md)

If the documentation does not answer your question, open a GitHub Discussion or an issue with enough context for the community to help.

## Bug Reports and Feature Requests

Use the GitHub issue templates to report reproducible bugs or propose features. Before opening an issue, search existing issues to avoid duplicates.

Please do not use issues for general support questions, private information, or security vulnerabilities.

## Security Issues

Report suspected vulnerabilities privately according to the [Security Policy](SECURITY.md). Do not disclose security issues publicly before they are reviewed.

## Community Conduct

All participants are expected to follow the [Code of Conduct](CODE_OF_CONDUCT.md).
